# Грандшеф — калькулятор маржи

Веб-инструмент для моделирования цен и маркетинговых сценариев по 4 каналам продаж: Wildberries, Ozon, Яндекс Маркет, собственный сайт.

## Открыть
- Локально: открой `index.html` двойным кликом
- Онлайн: https://yuriykslv.github.io/grandchef-margin-calc/

## Что внутри
Один HTML-файл, Tailwind подгружается с CDN, JS считает в браузере.

- Доли продаж по каналам — из факта продаж
- Учёт % выкупа (логика дельты от базового %)
- Распределение общего маркетингового бюджета по долям продаж
- Подсветка лучшего канала по чистой прибыли

## Формулы
Маржа на ед. = Цена до СПП − Себес − (Комиссия + Логистика + ДРР) − Хранение + Премия за СПП

Дельта выкупа = 2 × Логистика × ((1−выкуп)/выкуп − (1−база)/база)

Чистая прибыль = Σ (Маржа × штуки канала) − Маркетинг − Постоянные расходы
